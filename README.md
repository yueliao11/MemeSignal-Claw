# MemeSignal Claw 🦞
币安AI Agent大赛参赛作品：用OpenClaw + 币安技能实现的Meme猎手 + 交易信号双核助手

## 项目简介
帮助用户实时监控Meme热度 → 自动安全审计 → 结合交易信号给出行动建议。  
解决痛点：Meme追高易rug + 信号碎片化。

## 已用技能
- meme-rush  
- crypto-market-rank  
- query-token-audit  
- query-token-info  
- trading-signal  
- alpha  
- （可选）spot / futures / square-post
